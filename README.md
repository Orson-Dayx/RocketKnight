# RocketKnight
Yolo-coded game with ECS-pattern.

## Gameplay
A small game about a knight and modern technology, accompanied by a cloud of annoying bats.
## Control
Here is player control help-note
```
W,A,S,D - to move;
SPACE - to attack;
F12 - debug mode
```
